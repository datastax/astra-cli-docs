= {product}
:page-layout: landing

A fast, reliable command-line tool for working with {company} {product-short}.

The {product} gives you full control of your {product-short} resources directly from your terminal.
It's designed for developers and operators who prefer automation, scripting, and repeatable workflows over clicking through the {astra-ui} or interfacing directly with the APIs.

----
# Set up connection profiles
astra setup

# Manage databases
astra db create my_db -r us-east1

# Manage streaming tenants
astra streaming create my_tenant --region us-east1

# Run built-in utilities
astra db cqlsh start my_db
astra streaming pulsar shell my_tenant
----
